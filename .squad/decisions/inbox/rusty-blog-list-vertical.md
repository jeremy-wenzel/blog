# Rusty Decision: Vertical blog listing

- **Date:** 2026-05-17T09:48:04.314-07:00
- **Scope:** home page post preview and `/blog/` archive listing
- **Decision:** Use a single-column `.blog-card-list` for both listing surfaces, with the home page capped at 10 newest posts and `/blog/` showing the full archive.
- **Why:** The editorial/terminal look reads better as a vertical scan, gives each card more room for title + excerpt, and keeps the home page from feeling like a dense gallery.
